# 微信付费进群聊 & 红娘牵线服务平台

## 项目简介
这是一个专业的微信付费服务平台，提供两大核心服务：
- **微信付费进群聊**：高质量社交圈子，包括学习交流群、商务合作群、情感交流群
- **专业红娘牵线服务**：个性化婚恋匹配服务，帮助用户找到理想的另一半

## 技术特点
- 响应式设计，完美适配手机端和PC端
- 现代化的UI/UX设计
- 集成第三方支付系统
- 安全可靠的支付流程

## 支付系统配置

### 支付网关信息
- **网关地址**: `http://pay.jyyun.cn/submit.php`
- **商户PID**: `your_pid_here`
- **商户密钥**: `your_key_here`

### 重新对接API特性

#### 支付方式支持
- **支付宝支付** - 支持手机端和PC端
- **微信支付** - 支持扫码和H5支付
- **银行卡支付** - 支持主流银行快捷支付

#### 安全特性
- **签名验证** - 双重签名验证机制
- **参数校验** - 完整的支付参数验证
- **错误处理** - 友好的错误提示和重试机制
- **回调验证** - 支付回调签名验证

#### 技术实现
- **订单管理** - 自动生成唯一订单号
- **状态管理** - 完整的支付状态跟踪
- **异步处理** - 非阻塞支付流程
- **响应式设计** - 移动端优化支付体验

### 支付流程
1. 用户选择服务并点击支付
2. 系统验证支付信息完整性
3. 生成唯一订单号和支付签名
4. 显示支付方式选择界面
5. 构建支付参数并验证签名
6. 跳转到支付网关页面
7. 用户完成支付后返回网站
8. 处理支付回调并验证签名
9. 显示支付结果页面

### API接口说明

#### 支付初始化
```javascript
initiatePayment(service, amount)
```
- 参数验证和订单生成
- 支付方式选择初始化
- 支付弹窗显示

#### 支付处理
```javascript
processPayment()
```
- 支付参数构建
- 签名生成和验证
- 支付页面跳转

#### 回调处理
```javascript
handlePaymentCallback(callbackData)
```
- 回调签名验证
- 支付状态更新
- 结果页面跳转

## 使用方法

### 本地运行
1. 将所有文件放在Web服务器目录下
2. 通过浏览器访问 `index.html`
3. 测试支付功能（需要配置真实的支付网关）

### 部署到服务器
1. 上传所有文件到Web服务器
2. 配置支付网关回调地址
3. 测试完整的支付流程

## 支付回调配置

### 通知地址 (notify_url)
- 用于支付平台异步通知支付结果
- 需要配置为可公开访问的URL

### 返回地址 (return_url)
- 用户支付完成后跳转的页面
- 通常设置为支付成功页面

## 安全注意事项

1. **密钥安全**
   - 保护商户KEY不被泄露
   - 不要在客户端暴露敏感信息

2. **数据验证**
   - 验证支付回调的签名
   - 检查支付金额和订单状态

3. **HTTPS加密**
   - 生产环境必须使用HTTPS
   - 确保数据传输安全
